# Quran CLI Player

## Introduction
Quran CLI Player is a command-line interface application that allows users to listen to the Quran in a simple and efficient way. It provides users with the ability to play, pause, and navigate through Quranic verses.

## Features
- Play audio of the Quran from various reciters.
- Navigate by surah and ayah.
- Support for multiple languages.
- Lightweight and easy to use.

## Installation
To install Quran CLI Player, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Venvirus/quran-cli.git
   ```
2. Navigate into the project directory:
   ```bash
   cd quran-cli
   ```
3. Install the required dependencies:
   ```bash
   ./install.sh
   ```

## Usage
To start using Quran CLI Player, execute:
```bash
./quran-cli
```

Once the application is running, you can use the following commands:
- `play <surah>:<ayah>` - Play the specified verse.
- `pause` - Pause the current playback.
- `stop` - Stop the playback.
- `next` - Play the next verse.
- `prev` - Play the previous verse.
- `help` - Display a list of available commands.
